
# Open Nerse Web

Hello, say hello to what is behind the Open Nerse website!


## Run Locally

Clone the project

```bash
  git clone https://github.com/openerse/openerse-web
```

Go to the project directory

```bash
  cd openerse-web
```

Install dependencies

```bash
  yarn install
```

Start the development server

```bash
  yarn dev
```
